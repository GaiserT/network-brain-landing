# Greencore Sales Brain landing

Public landing package for Vercel.

## Files

- `index.html` — public landing page.

## Publishing rules

- Publish only this folder, not the full HQ.
- Do not publish `AGENTS.md`, `me.md`, `now.md`, PRD drafts, private notes, tokens, keys, or working customer data.
- Vercel settings: framework `Other`, no build command, root/output is the repository root.
- Update flow: edit `index.html`, commit, push to `main`; Vercel auto-deploys the new version.
