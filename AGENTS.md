# NETWORK BRAIN LANDING AGENTS

## Вход и границы

Прочитать `README.md`. Это публичный репозиторий лендинга. В нем допустимы только материалы, которые можно публиковать. Не сохранять сюда внутренние решения, handoff-файлы, приватные заметки, цели HQ, CRM-данные, секреты или рабочие обсуждения.

## Завершение сессии

Канонический межконтурный протокол при локальной работе: `../HQ/ops/knowledge-capitalization.md`.

Содержательная работа завершается проверяемым изменением лендинга, обновлением связанной канонической задачи или явным объяснением отсутствия долговечного результата.

Если во время работы появилось системное знание, создать передачу системного знания в приватном `../network-brain/handoffs/system-learning/YYYY-MM-DD-network-brain-<slug>.md` по шаблону `../HQ/ops/templates/system-learning-handoff.md`. Если приватный соседний репозиторий недоступен, не записывать передачу в этот публичный репозиторий: показать ее владельцу и явно отметить, что маршрутизация в Network Brain / System Evolution не завершена.
